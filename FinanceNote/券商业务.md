# 账户关系
## 一码通账户
中国证券登记结算公司给的，只能有一个

## 资金账户
券商给的


## 证券账户（股东账户）
由交易所给的

股东账户是投资者进行证券交易的核心账户。 
沪深两市账户区别： 沪市（上海）A股股东账户：每人最多可开设三个，需分别在不同券商办理。

深市（深圳）A股股东账户：虽同样每人最多开设三个，但支持多个券商共用一个账户，因此投资者可以在多家券商开户。 股东代码为10位数字

用途：该账户用于买卖股票、基金、债券等证券。


| 账户类型      | 核心功能                                    | 数量关系（关键规则）                                               |
| --------- | --------------------------------------- | -------------------------------------------------------- |
| **一码通账户** | **总管家/总账本**：把你名下所有券商的账户都关联起来，汇总你的全部资产   | **严格的一对一**：你作为一位投资者，**全国范围内只能有一个**一码通账户                  |
| **证券账户**  | **子仓库/分类账**：记录你在特定市场（如上海A股、深圳B股）持有的具体证券 | **一对多**：你的“总账本”下，可以关联**多个**子账户。例如，可以同时有沪A、深A、北交所等不同市场的账户 |
| **资金账户**  | **钱包/交易通道**：在券商这里用来存放资金、进行买卖结算的“钱包”     | **多对一**：你可以在**同一家券商**里，绑定**最多5个**不同银行的银行卡作为“辅钱包”         |
