# 一 . 工作原理

## 工作区-从零开始

```shell
git init 

# 创建**新的空仓库**（从零开始）
    
```
 创建一个.git 

## 暂存区

```shell
git add 
```

## 本地仓库

```
git commit 
```


将文件保存在本地仓库


## 推送至远程仓库

```shell
# 连接远程仓库

git push 
```
