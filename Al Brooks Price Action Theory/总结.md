

# 一. 独立的市场阶段分类


## 1. 突破

只顺势，看情况加仓，移动止盈，让利润奔跑

## 2. 窄通道

只顺势，
看情况加仓，
移动止盈，让利润奔跑，
止损放在回调低点


## 3.  宽通道

顺势做波段

## 4. 震荡区间

剥头皮，高抛低吸

太窄的震荡区间不做，容易大亏

在震荡区间一直剥头皮，必然会遇到突破，会止损一次。

| 形态类型     | 预估时间占比        | 对应的市场状态与心理                                                                                                                             |
| -------- | ------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| **震荡区间** | **60% - 70%** | **市场的“待机”状态**。大部分时间，多空双方都在试探、休息或吸筹，没有明确方向。这是市场的常态，也是 Brooks 所说的“让大多数交易者感到困惑”的阶段[](https://www.laohu8.com/post/423953777160200?page=2)。 |
| **宽通道**  | **15% - 25%** | **趋势的“分歧”状态**。趋势已经确立，但分歧加大，回调加深。虽然看着像趋势，但内部已经开始剧烈摇摆。                                                                                   |
| **窄通道**  | **5% - 10%**  | **趋势的“单边”状态**。这是交易员的“甜蜜区”，但极其稀缺。一天中可能只有1-2段这样的流畅行情，通常在早盘或午盘某个时段突然爆发并快速结束。                                                              |
| **突破**   | **< 5%**      | **趋势的“瞬间”状态**。严格意义上的“突破”只是一两根K线的事情。它不是一个“时间段”，而是一个“时间点”或极短的过程，是市场情绪的爆发瞬间[](https://www.laohu8.com/post/423953777160200?page=2)。        |

# 二. 阶段演化

突破——窄通道——宽通道——震荡区间

这是一个标准的市场价格行为的演变过程

但大多数是不标准的

## 1. 交易区间演变

### 1.1  交易区间 ↔ 突破 (TR → B/O)
- **演变**：价格在交易区间内来回运动，最终某一天（或某一根K线）以强劲的力量**突破**区间的边界。这个“突破”宣告了交易区间的结束和新状态的开始。

#### 如何做这个突破行情？

追进去后，如果行情回撤到成本位，再等一等，说不定有大行情，如果没有就小亏离场
最好用突破条件单，市价单容易亏

## 2. 突破演变

#### 2.1 突破 ↔ 窄通道 (B/O → NCT)

价格**突破**交易区间后，没有回头，而是以一连串实体饱满、回调很小的K线，沿着一个狭窄的通道持续前进。这就形成了**窄通道**。

## 3.窄通道演变

####  3.1 窄通道  →  宽通道 (NCT → WCT)
- **演变**：原本流畅的**窄通道**开始出现长长的影线、大幅的回调，K线实体变小且排列不再紧凑。通道的上下轨开始张开，形成一个“喇叭口”或平行但宽阔的通道，即**宽通道**。
    
- **布鲁克斯的视角**：宽通道表示趋势仍在，但多空分歧加大。买方不再那么坚决，卖方开始反击。此时，交易难度增加，最佳策略通常是从“持有”转为“高抛低吸”，或者在宽通道的上轨和下轨附近进行逆势交易，但必须快进快出。

## 4.宽通道演变
### 4.1宽通道 → 交易区间

这是趋势衰竭，市场重新进入平衡的过程。

- **演变**：**宽通道**的震荡越来越剧烈，最终价格既创不了新高（或新低），也破不了新低（或新高）。通道的边界变得模糊，价格开始在一个水平区域内无序运动。这就重新演化成了**交易区间**。
    
- **布鲁克斯的视角**：此时，之前清晰的趋势结构已经瓦解。市场再次进入“无人区”，交易者应停止使用趋势策略，转用区间交易策略，或者干脆观望，等待下一次突破。
